Accessing PostgreSQL:

1. sudo -i -u postgres
2. psql
 
Use Postgres commands to create DB and Tables and use /q to exit

POSTGRESQL COMMANDS

SHOW and USE does not work

Command,Action

\l,   List all databases.

\c    db_name,Connect to a specific database (Like USE in MySQL).

\dt,  Describe Tables: List all tables in the current database.

\d    table_name,"Describe Table: Show columns, types, and constraints for a table."

\du,  List Users: Show all database users/roles.

\df,  List Functions: Show all stored functions.

\q,   Quit: Exit the PostgreSQL terminal.

\! clear, Clear the terminal screen.
